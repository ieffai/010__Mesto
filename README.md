# 010__Mesto
Interactive page where you can add photos, delete them and to like them 
 
*latest ver. v 0.0.1*

## Used technologies
- JS;
- CSS;
- HTML;
- WebPack;
- GIT.

## Demo link

https://ieffai.github.io/010__Mesto/

## Local deploy

1. Clone repository
2. Install missed npm modules
```
npm install
```
3. Rul local host
```
npm run dev
```
